功能描述：  
1.用户的注册、登录、退出系统  
2.用户的个性化推荐（基于物品的协同过滤）  
3.家政人员上传资料、身份认证和资格认证  
4.消费者发布预约、查看预约、确认预约、评价订单、删除订单、修改个人资料  
5.家政人员修改个人资料、申请预约、接受订单、取消订单、完成订单  
6.管理员对用户认证管理和系统的量化查看
